Problem 1

Let us define \[\Sigma = \phi^a+\phi^b+\phi^c\] where \(\phi=\frac{1+\sqrt5}{2}\) is the golden ratio. We wish to show that when \(\Sigma\in\mathbb{Z}\) either \(\Sigma=1, 5\) or \(n^2 \pm 2\) for some \(n\in\mathbb{Z}\). To establish these results we will first determine a closed form expression for \(\phi^n\).

Lemma 1.1

For any integer \(n\), not neccessarily positive, \(\phi^n=F_n\phi+F_{n-1}\) where \(F_n\) is the \(n^{th}\) Fibonacci number defined by \(F_{n+2}=F_{n+1}+F_n\) and initial conditions \(F_1=F_2=1\)

Proof

To prove the above identity it is useful to note that \(\phi\) satisfies \[\phi^2=\phi+1\] This establishes the identity for \(n=1\) and furthermore by multiplying both sides of the equation by \(\phi^n\) we get that \[\phi^{n+2}=\phi^{n+1}+\phi^{n}\] Therefore, if we assume the identity for all \(k\le n+1\) \[\phi^{n+2}=F_{n+1}\phi+F_n+F_{n}\phi+F_{n-1}=F_{n+2}\phi+F_{n+1}\] By induction the result is true for all positive \(n\). The result also holds for negative \(n\) since powers of \(\phi\) satisfy the same reccurence relation as the fibonacci numbers. \(\blacksquare\)

In addition to the above lemma we also need to know how the fibonacci numbers behave when the indices are negative.

Lemma 1.2

The following identity holds \[F_{-n}=(-1)^{n+1}F_n\]

Proof

To establish the base case note that we must have \(F_0=0\) and thus \(F_{-1}=1\). Therefore \(n=1\) holds. Now assume the identity holds for all \(k\le n\). Then \[F_{-(n+1)}=F_{-n-1}=F_{-n+1}-F_{-n}=(-1)^nF_{n-1}+(-1)^nF_n=(-1)^nF_{n+1}\] The result follow by induction on \(n.\blacksquare\)

We can now rewrite \(\Sigma\) as \[\Sigma=(F_a+F_b+F_c)\phi+F_{a-1}+F_{b-1}+F_{c-1}\] Since \(\phi\) is irrational, \(\Sigma\) is an integer iff \(F_a+F_b+F_c=0\). This cannot be the case if \(F_a,F_b,F_c\) are all positive so that leaves the following three cases to consider (up to a reordering of the variables):

  1. \(F_a,F_b>0\text{ and } F_c\le0\)
  2. \(F_a>0\text{ and } F_b,F_c\le0\)
  3. \(F_a,F_b,F_c\le0\)

Before we proceed however, we present in the following lemma the solutions to the equation \(F_i+F_j=F_k\) which we will need later.

Lemma 1.3

If \(F_i+F_j=F_k\) for non-negative indices \(i,j,k\) with \(i\le j\) then \((i,j,k)\) is either of the form \((i,i+1,i+2)\) or one of \((0,0,0),(1,1,3),(2,2,3)\).

Proof

The fibonacci numbers are monotone increasing for positive indices so \[F_j< F_k \implies j+1 \le k\] It follows that \(F_i+F_j\ge F_{j+1}\) and so we reach the following \[F_{j-1}\le F_i\le F_{j}\] We therefore need only consider the possibilities \(i=j \text{ and } j=i+1\). If \(j=i+1\) the case \((i,i+1,i+2)\) follows immediately. If \(i=j\) then we have \(2F_i=F_k\) and thus \[F_i\le F_k = 2F_i\le F_i+F_{i+1}=F_{i+2}\implies F_i\le F_k\le F_{i+2}\] This means \(k=i, k=i+1\) or \(k=i+2\). If \(k=i\) then \(2F_i=F_i\implies i=j=k=0\). If \(k=i+1\) then \[2F_i=F_{i+1}\implies 2F_i=F_i+F_{i-1}\implies F_i=F_{i-1}\]Since the fibonacci numbers are strictly increasing for indices \(\ge 2\) the only possibility is \(i=j=2\) and \(k=3\). Similarly the case \(k=i+2\) gives the last possibility \((1,1,3)\). \(\blacksquare\)

We are now ready to tackle the different cases.

1. To get \(F_a+F_b+F_c=0\) when \(F_a,F_b>0\text{ and } F_c\le0\) we must have that \(c\) is negative and even by lemma 1.2. Write \(c=-2k,k\in\mathbb{N}\). Then \[F_a+F_b=F_{2k}\] By lemma 1.3 the only possibility is that \(a=2k-2\) and \(b=2k-1\). Therefore \[\Sigma=F_{a-1}+F_{b-1}+F_{c-1}=F_{2k-3}+F_{2k-2}+F_{-2k-1}=F_{2k-1}+F_{2k+1}\]Considering the first few values of \(k\) we find that \[\begin{align*} k=1, & \quad \Sigma=3=1^2+2 \\ k=2, & \quad \Sigma=7=3^2-2 \\ k=3, & \quad \Sigma=18=4^2+2 \\ k=4, & \quad \Sigma=47=7^2-2 \\ k=5, & \quad \Sigma=123=11^2+2 \\ k=6, & \quad \Sigma=322=18^2-2 \end{align*}\] In particular upon inspection we conjecture that \[F_{2k-1}+F_{2k+1}=(F_{k-1}+F_{k+1})^2-2\cdot (-1)^{k}\]

Proof

By inspection of the above table the identity holds for \(k=1\). Now, if we assume it is true for all \(k\le n\) then \[\begin{align*} F_{2(k+1)-1}+F_{2(k+1)+1} & =F_{2k+3}+F_{2k+1} \\ & = F_{2k+3}+ (F_{k-1}+F_{k+1})^2-2\cdot (-1)^{k}-F_{2k-1} & = \end{align*}\] But by the identity \(F_{n+m}=F_{n-1}F_m+F_nF_{m+1}\)
